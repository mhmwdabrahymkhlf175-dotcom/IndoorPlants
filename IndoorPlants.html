About This Project

IndoorPlants Analyzer helps users detect plant diseases using images. It provides treatment tips and care instructions. Built with HTML, Tailwind CSS, and JavaScript for interactive experience.

The project aims to make plant care accessible and easy, combining modern UI design with practical functionality.
